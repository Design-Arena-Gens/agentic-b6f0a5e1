# Beton

Beton is a Next.js 14 experience site highlighting intelligent concrete systems, sustainable supply chains, and digital pour management for infrastructure teams.

## Getting Started

- `npm install`
- `npm run dev` → local development at `http://localhost:3000`
- `npm run build` → create an optimized production build
- `npm run start` → serve the production build locally
- `npm run lint` → lint with Next.js Core Web Vitals rules

## Deployment

The project is optimized for Vercel. Build the production bundle before deploying to surface issues early.
